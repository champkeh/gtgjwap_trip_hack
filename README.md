## 目录结构说明
- `src` - 从网站提取出的源文件
- `ajcs.js` - 计算`sid`参数的模块
- `stationQuery.js` - 站站查询接口的sid
- `trainOntime.js` - 车次查询接口的sid
- `dynamicQuery1.js` - 按照航班号查询接口的sid
- `dynamicQuery2.js` - 按照起降地三字码查询接口的sid

更多接口可参看`stationQuery.js`和`trainOntime.js`
